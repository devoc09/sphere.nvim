## Install
### Using `lazy.nvim`
```lua
{ 'devoc09/sphere.vim' }
```

### Using `vim-plug`
```vim
Plug 'devoc09/sphere.vim'
```

## Config
Inside `init.vim`
```vim
colorscheme sphere
```

Inside `init.lua`
```lua
vim.cmd('colorscheme sphere')
```
