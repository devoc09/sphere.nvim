## Install
### Using `lazy.nvim`
```lua
{ 'devoc09/sphere.nvim' }
```

### Using `vim-plug`
```vim
Plug 'devoc09/sphere.nvim'
```

## Config
Inside `init.vim`
```vim
colorscheme sphere
```

Inside `init.lua`
```lua
vim.cmd('colorscheme sphere')
```
